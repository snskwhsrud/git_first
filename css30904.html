css (cascading style sheet) 웹 페이지 스타일 정의( 색상, 레이아웃, 폰트 등 ) 1. 선택자 (selectors) : Css 셀럭터는 스타일을 적용할 HTML 요소를 식별하는데 사용한다 (ID, CLASS) 2. 속성과 값 (Properties & Values) : 선택한 요소에 적용할 스타일은 (속성 : 값) 정의된다 color : red; 3. 선언 블록 : 한 개 이상의 CSS 코드를 중괄호{}로 감싼 것이다 속성과 값으로 이루어진 값들이 들어있다 4.Rule set : 셀렉터와 선언 블록을 합친 것이다 특정 HTML 요소에 여러 CSS요소를 적용할 떄 사용한다 5. 상속(Inheritance): 부모 요소와 자식 요소로 상속된다 우선순위(Specificity) 와 캐스캐이딩(Cascading) : 여러 스타일을 같은 요소에 적용될 때, 우선 순위와 Cascading 규칙에 따라 어떤 스타일이 먼저 적용될지 결정한다 7. 박스 모델 ( Box Model) : 모든 HTML 요소는 사각형 박스로 간주하며, 이 박스는 여백(margin), 테두리(border) ,패딩( padding). 내용(content) 영역으로 구분되어 있다 8.레이아웃(Layout) : 페이지 내부 레이아웃을 설정한다 어떻게 배치할 것인가 (템플릿) 9. 미디어 쿼리(Media Queries) : 특정 조건에 따른 스타일을 적용하여, 반응형 웹 디자인을 할 수 있다 10. 애니메이션과 전환 (Animation&Transition): 상태 변경 부드럽게 표현할 수 있도록 제공한다 CSs 스타일 시트를 사용하면, 개별 HTML 요소에 직접 적용하는 대신 한 곳에서 스타일은 전부 관리할 수 있다 1. 외부 스타일 시ㅌ(External Stylesheet) : 별도의 .CSS 파일에 CSS 코드를 작성하고, HTML 문서에서 태그를 통해 파일을 참조하여 사용하는 방식이다 이렇게 사용하게 되면, 여러 HTML에서 같은 CSS 파일을 적용 가능하다 2. 내부 스타일 시트 (Internal stylesheet) : HTML 문서 내부에 태그를 사용하여 직접 CSs코드를 작성한다 이 방법은 해당 HTML 문서에서만 필요한 특정 스타일이 있을 수 있다 3. 인라인 스타일 (Inline style) : HTML 요소의 style 속성 내부에 직접 Css 코드를 작성하는 방식
