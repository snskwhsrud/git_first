-인라인 프레임(Inline Frame) iframe HTML 요소를 다른 웹페이지나 현재 웹페이지에 삽입하는데 사용이 된다. 현재 페이지 일부 영역에서 다른 문서를 표시하고 상호작용을 하며 독립적인 컨테이너 역할을 한다. src = 문서 ur1 width, height: 사이즈 조절 px 또는 %단위로 지정 1. 문서 삽입 : src 속성을 활용하여 다른 웹페이지, 문서, 이미지, 동영상 등 외부 리소스를 삽입할 수 있다 2. 독립성 : 독립적인 컨테이너를 가진다 iframe 내부에 로드된 문서는 외부 문서와 격리되어 있다 (충돌방지) 3. 크기 조절 : 4. 상호작용 : iframe, 다른 문서들이 상호작용이 가능하다 5. 충돌 및 성능 고려사항 : iframe 너무 많이 사용하게 되면, 성능 저하가 일어난다 부모, 자식 프레임 충돌발생 주의 6. 보안 고려사항 : coRs 정책으로 일부 사이에서는 다른 도메인의 리소스가 인라인 프레임으로 로드되지 않을 수 있다 - 문서 구조화 시맨틱 태그(semantic tag)를 사용을 해서 웹 문서의 구조와 의미를 전달한다. 시맨틱 태그는 콘텐츠의 의미와 역할을 설명하고, 검색 엔진 최적화(SEO)와 웹 접근성을 향상시킨다 Htnl5에서 사용하는 시맨틱 태그들은 의미를 가지고 역할과 목적에 따라 사용된다. 웹 개발자가 페이지 구조를 이해하기 쉽고, 유지보수가 용이한 코드를 작성할 수 있다 검색 엔진은 시맨틱 태그를 통해 페이지와 구조를 파악해서 검색 결과에 반영할 수 있다 배리어프리(barrier-free) 서비스로 만들어 장애인, 사회적 약자들도 시맨틱 태그도 작성된 웹페이지 정보에 대한 접근성을 개선할 수 있다
